# Bear-Collecting-Fishes-Game-Java-Game
A simple Java game built with Greenfoot where the player controls a bear that collects fish while avoiding obstacles.

Features:

Player-controlled bear character with keyboard movement

Fish objects to collect for scoring

Obstacles that reduce score or end the game

Simple background, sound, and sprite-based animations

Built with Java and Greenfoot, demonstrating object-oriented game logic

How to Run:

Download and install Greenfoot (https://www.greenfoot.org/download
)

Open the project folder BearCollectingFishes in Greenfoot

Press the Run button to start the game

Technologies Used:

Java

Greenfoot IDE

OOP principles (inheritance, classes, objects, methods)
